# pharmfact
ФармФакт — телеграм-бот, который помогает проверить эффективность и безопасность лекарственных средств. Пользователь вводит название препарата и получает данные о степени доказанности, допустимых дозировках для взрослых, а также частоте приёма. Проект реализован на Java 21 с использованием Spring Boot и Telegram API.
